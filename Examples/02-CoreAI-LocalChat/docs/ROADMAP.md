# Roadmap

## Phase 1

- SwiftUI chat app shell.
- Mock runtime.
- Bundled model manifest.
- Model list and model detail screens.
- Active model selection.
- Manual `.aimodel` file detection.

## Phase 2

- Download manager.
- Application Support model storage.
- Download state, cancellation, retries, and integrity checks.
- Real Core AI LLM generation integration once the exact API surface is finalized.

## Later Phases

- Full settings screen.
- Prompt templates.
- RAG pipeline.
- Conversation persistence.
- Model compatibility validation.
